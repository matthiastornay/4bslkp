# friendly-superlattice-simulator
Deploy a web interface to solve custom III-V superlattice band structures in a navigator

Dans anaconda prompt, on initialise le serveur avec :
python -m http.server 8000

(ou seulement python -m server si le port a déjà été ouvert dans la même session)
Et on y accède avec : http://localhost:8000/Desktop/solver_kp_online/new_architecture/
